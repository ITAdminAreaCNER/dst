Guide: Google Docs for Service Work

The foundation for meeting minutes, agendas, and committee reports.

Why Use Google Docs?

Live Collaboration

Multiple members can write and edit a document at the same time.

Version History

Easily see who made changes and when, and revert to old versions if needed.

Accessible Anywhere

Access your documents from any computer or phone with an internet connection.

Easy Sharing

Share a link instead of sending attachments that can become outdated.

A Critical Note on Document Ownership

To ensure service records are preserved for future trusted servants, any document related to service (minutes, agendas, reports) must be created inside a Shared Drive. A document created in a personal "My Drive" is owned by the individual. If they leave service, that document and its history can be lost forever.

Common Service Use Cases

1. Meeting Minutes & Agendas

This is the most common and powerful use for a committee secretary. It ensures minutes are captured accurately and distributed easily.

  • Create in the Right Place: Always create the new document inside the correct Shared Drive folder (e.g., `01 - Meeting Minutes & Agendas`).
  • Use a Naming Convention: Name files consistently so they are easy to find and sort, for example: `2025-08-15 District 10 Minutes.docx`.
  • Powerful Shortcut: In a new Doc, type the `@` symbol and select "Meeting notes" from the menu. If you have a corresponding Google Calendar event, it will automatically pull in the date, title, and attendees.

2. Drafting Committee Reports & Documents

When a committee needs to draft a report, flyer, or set of guidelines, Google Docs allows for effective collaboration without creating dozens of different file versions.

Using Comments

Highlight any part of the text and click the "Add comment" icon that appears on the right. This is perfect for asking questions or giving feedback without changing the document itself.

Using Suggesting Mode

Change the mode from "Editing" to "Suggesting" at the top of the screen. Any changes you make will appear as colored suggestions that the document owner can approve or reject. This is the best way to propose changes to a draft.

3. Best Practices for Google Docs

  • Store in Shared Drives: This is the golden rule. A document related to service work must live in a Shared Drive, not a personal My Drive.
  • Use Templates: Create a template for your meeting minutes with a standard header and layout. Each month, just make a copy of the template.
  • Share with a Link: Instead of emailing PDF attachments, share the link to the Google Doc. This ensures everyone is always looking at the most current version.
  • Manage Permissions: Not everyone needs to be an editor. For a draft agenda sent out before a meeting, give members "Commenter" access so they can make suggestions without changing the document directly.
